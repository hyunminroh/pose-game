# 게임 이름: Fruit Catcher (과일 받기 게임)

## 1. 기본 구조
- 화면은 **3개의 구역 (LEFT / CENTER / RIGHT)** 으로 나뉩니다.
- 플레이어는 포즈를 통해 하단의 **바구니**를 이동시킵니다.
    - ⬅️ 왼쪽 포즈 (Left) → **LEFT** 구역으로 이동
    - ⬆️ 중앙 포즈 (Center) → **CENTER** 구역으로 이동
    - ➡️ 오른쪽 포즈 (Right) → **RIGHT** 구역으로 이동

## 2. 아이템 종류 (하늘에서 떨어지는 물체)
| 아이템 | 이미지(예시) | 점수 | 효과 |
| :--- | :---: | :--- | :--- |
| **사과** | 🍎 | +100점 | 기본 득점 아이템 |
| **포도** | 🍇 | +300점 | 획득 시 보너스 점수 |
| **폭탄** | 💣 | - | **게임 오버** (즉시 종료) |

## 3. 게임 진행 규칙
- **제한 시간**: 60초 (또는 무제한 + 라이프 시스템)
- **레벨 시스템**: 
  - 10초마다 레벨이 올라갑니다.
  - 레벨이 오르면 아이템이 떨어지는 속도가 **1.2배** 빨라집니다.
- **게임 오버 조건**:
  - 폭탄(💣)을 받았을 때

## 4. UI 요소
- **상단**: 현재 점수, 남은 시간
- **중앙**: 게임 화면 (플레이어 웹캠 + 떨어지는 아이템)
- **하단**: 플레이어 상태 (현재 인식된 포즈 표시)
